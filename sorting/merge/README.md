# Merge Sort

Merge Sort is type of sorting algorithm that follows divide-and-conqure paradigm. It was invented by johb von neumann in 1945.
This algorithm works by dividing an unsorted array into n partitions. each containing one element(A list of one element is consided sorted), then repeatedly merging partitions to produce new sorted list until there is only 1 sorted list remaining. this resulting list is fully sorted list.

The process of dividing the list is done recursively util is hits the base case of a list with one item.

Merge sort has a time complexity of O(n log n) for all cases (best, average and worst), which makes it highly efficient for large data sets.
